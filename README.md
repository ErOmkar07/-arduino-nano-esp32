# Arduino Nano ESP32 Resources

This repository contains useful resources for the Arduino Nano ESP32 board.

## Files Included

- Datasheet PDF
- Full Pinout PDF
- Schematics PDF
- CAD Files
- STEP Files

## Purpose

This repository is created for:

- Hardware design reference
- PCB development
- Embedded systems projects
- ESP32 learning and development

## Board Information

The Arduino Nano ESP32 combines the simplicity of Arduino with the power of the ESP32 microcontroller.

Features:
- WiFi and Bluetooth support
- Compact Nano form factor
- Low power operation
- Suitable for IoT projects
